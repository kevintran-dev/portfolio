About Me

Projects

Digimon Deck Builder

Browse Digimon cards and create decks out of these cards. Frontend created with React. Backend created with Express. PostgreSQL used for the database. Key features include creation of decks, user authentication. Challenges like protecting sensitive data and and saving deck information were tackled. Future plans include card counter. Its development underscored state management, and a strong backend/database foundation.

Personal Board Game Library

Create a catalog of board games by scanning the barcode of the game. Created with Python and Flask. Uses SQLAlchemy to store board game information to the database. Retrieves board game names by UPC from GameUPC API, and board game information from BoardGameGeek API.
